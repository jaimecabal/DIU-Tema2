GOMS Y PATHFINDER


GOMS

GOMS es un modelo especializado de procesar información humana para la interacción persona ordenador que describe la estructura cognitiva de un usuario en cuatro componentes. GOMS constituye de un “a set of Goals, a set of Operators, a set of Methods for achieving the goals, and a set of Selections rules for choosing among competing methods for goals”; es decir, un set de Metas, Operadores, Métodos para conseguir dichas metas y Reglas de selección para escoger como se completan dichas metas.

Pathfinder

El pathfinder es una estrategia para el control de información y la gestión del conocimiento. Es el tipo de red semántica que hemos visto. Su objetivo es analizar las distintas relaciones entre conceptos.
Este procedimiento representa los conceptos en nodos de una red, y las relaciones semánticas entre los conceptos como punteros que unen estos nodos.
Las palabras más importantes suelen ser las que tienen el mayor número de relaciones.
